# Daily Dashboard

## `About`

This application is a dashboard to track your daily routine, tasks, and set categorized deadlines or goals for yourself.

## `Goals`

The goals for this project are the following:

1. Create a tool for me to organize my items into
2. Hone my skills in fullstack application development
3. Build up my portfolio

## `AI Usage`

I will be using AI like `Claude` and `CoPilot` in this project to assist in the building process. This does **_NOT_** mean I am vibe-coding, I am merely using it as an assistant to my own skills.
